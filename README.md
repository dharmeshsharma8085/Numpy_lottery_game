# Numpy_lottery_game
A NumPy-based lottery simulator that generates participants, selects random lucky numbers, finds winners, and calculates winning probability.

# NumPy Lottery Simulator

A simple lottery simulator built using NumPy.

## Features
- Generates 10,000 lottery participants
- Selects 10 unique lucky numbers randomly
- Finds winners using NumPy array operations
- Displays total winners
- Calculates winning probability
- Calculates prize per winner
- Shows unique winning numbers

## Technologies Used
- Python
- NumPy

## Concepts Practiced
- Random Number Generation
- NumPy Arrays
- Boolean Indexing
- Conditional Statements
- Probability Calculation
